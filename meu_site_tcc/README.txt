Instruções rápidas:
1) Instale XAMPP (ou similar) com Apache + MySQL + PHP.
2) Copie a pasta 'meu_site_tcc' para o diretório 'htdocs' do XAMPP (ex: C:\xampp\htdocs\meu_site_tcc).
3) Inicie Apache e MySQL pelo painel do XAMPP.
4) Em http://localhost/phpmyadmin, importe o arquivo sql/banco_tcc.sql para criar o banco e tabelas.
5) Abra no navegador: http://localhost/meu_site_tcc/index.php
6) Abra includes/db.php e ajuste as credenciais do MySQL se necessário.
7) As páginas principais já são responsivas com Bootstrap. Para testar, crie usuários e instituições via phpMyAdmin ou crie formulários adicionais.
8) Observações: logo.svg está em assets/logo.png (SVG string). Substitua por imagens reais se desejar.
